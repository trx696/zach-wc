How WC reimbursement works

In Workers' Comp, the Manufacturer NDC sets the reimbursement.

Original Manufacturer NDC→ Sets the AWP→ Billed per Fee Schedule→ Commission (% of collected)

Repackager NDCs generally aren't reimbursed in Workers' Comp; the original-manufacturer's NDC is what gets paid. Terrain's products use original-manufacturer NDCs, so the dispensed NDC determines the billed value on scripts already being written.
